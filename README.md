# Spotify Favorite Dashboard

## install

`npm install`

## Launch

`npm run dev` and open in your browser `http://localhost:5173`
For your first launch, you will need to create a *.env* file, and  fill it such as :
```
SPOTIFY_CLIENT_ID=123
SPOTIFY_CLIENT_SECRET=123
```
and run `npx ts-node src/scripts/fetchSpotify.ts`
